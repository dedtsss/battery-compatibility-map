# Карта взаимозаменяемости аккумуляторов электроинструмента

Интерактивная карта и таблица совместимости аккумуляторов электроинструмента по материалам темы форума 78294.ru.

## Открыть карту

После включения GitHub Pages карта будет доступна по адресу:

https://dedtsss.github.io/battery-compatibility-map/

## Что внутри

- прямая совместимость аккумуляторов и инструмента;
- совместимость внутри платформ и семейств;
- варианты через переходники;
- случаи, где нужна доработка: подпил, перемычка, термодатчик и т.п.;
- известные несовместимости;
- случаи, где аналогов не найдено;
- пометки по подтверждению: текст / текст + фото.

## Файлы

- `index.html` — интерактивная карта для GitHub Pages.
- `data/relations.csv` — таблица связей.
- `forum-post.md` — готовый текст сообщения для форума.

## Важно

Это не официальная таблица производителей, а структурированная выжимка из практических сообщений пользователей форума. Перед покупкой или переделкой нужно самостоятельно сверять форму разъёма, защёлки, полярность, силовые и дополнительные контакты, термодатчик / ID-контакт, токовую отдачу АКБ и работу защиты от переразряда и перегрева.

Переходник часто даёт только механическое и электрическое подключение. Это не равно полной штатной совместимости.
